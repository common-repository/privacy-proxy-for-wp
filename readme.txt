== Description ==
Privacy Proxy is a simple, effective, and state-of-the-art anonymization solution for businesses and organizations that wants to continue using Google Analytics while complying with GDPR. If you need information about why there is problem with using Google Analytics without further measures please visit our FAQ page about the subject. 

The Privacy Proxy anonymizes data before it is sent to Google, such that Google cannot track the persons behind the web page visits. In doing so, we use state-of-the-art anonymization technology.

== Installation ==
1) Activate the plugin. 
2) Insert the two values ("host id" and "site tag") from PII Guard.
3) Test that your site is not sending Google Analytics information to servers at Google from https://piiguard.com/ga-scanner.

== Frequently Asked Questions ==

Remember to remove the inclusion of the components which Privacy Proxy replaces (e.g. Google Analytics).
